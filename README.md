# AWS Professional DevOps certificate prep material

Material found in this repo is created from various sources, most notably:

* [The AWS Certified DevOps Engineer course by Adrian Cantrill](https://cantrill.io/)

```Bash
# Create a book
mdbook init pilot-book

# Update and add files then build the book
mdbook build
# This will generate a directory named book which contains the HTML content of your book. You can then place this directory on any web server to host it.

# Open the book locally
mdbook serve --open
```
